## Где остановились

M2 закрыт полностью шестью feat-коммитами через две сессии 2026-05-11: `/demos/live-session` — `d73130d` каркас + dual RPM, `b32619c` boost/IMU/GPS/current-values, `241c54a` ticker + signed-block bar; `/demos/anti-cheat-replay` — `1fc8508` ScrubTimeline + RPM violation-bands, `662fe21` boost/speed-throttle/IncidentSummary с click-to-seek, `9debd83` HashChainViz + PlayControls с jump-to-violation. Новый публичный API mockData — `VIOLATION_WINDOWS: Record<SessionId, ViolationWindow[]>` (точные диапазоны из generator, не приближения вокруг incident.tMs). Все коммиты — tsc чисто, dev-сервер чистый, curl-smoke HTTP 200 + ключевые лейблы в SSR. Браузерно-визуально пользователь подтвердил только live-session на :3001; replay визуально пользователем не проверялся. Дорога открыта в M3.
