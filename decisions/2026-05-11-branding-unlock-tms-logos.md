# Branding unlock — TMS логотипы и фирменный оранжевый

**Дата:** 2026-05-11
**Контекст:** M3 — старт работ над парком и контекстными экранами

## Контекст

В ROADMAP в «Открытые вопросы → Критичные» висел пункт «Брендинг Telos» с подпунктом «Сейчас в Navigation.tsx — красный квадрат-плейсхолдер. Без бренд-материалов делается нейтральный technical look, потом перекрашивается». Решение `decisions/2026-05-10-defer-dark-theme-until-branding.md` явно отложило тёмную палитру до фиксации брендинга.

В ходе обсуждения дайно-данных пользователь указал на смежную папку `/Users/mactm/Projects/presentation.md/` (отдельный Vite-проект корпоративной презентации TMS), где лежат официальные логотипы: `Логотип TMS_графит_eng.svg` и `Логотип TMS_белый_eng.svg`. Из SVG `defs` извлечены два фирменных цвета: графит `#39393f` и оранжевый `#ff4f00` (вертикальная разделительная полоска в логотипе).

## Решение

Брендинг частично разморожен. Конкретно:

1. SVG скопированы в `public/` под латинскими именами `tms-logo-graphite.svg` и `tms-logo-white.svg`.
2. В `globals.css` через `@theme inline` зафиксированы два бренд-цвета: `--color-tms-orange: #ff4f00` и `--color-tms-graphite: #39393f`. Доступны как Tailwind-классы `bg-tms-orange`, `text-tms-graphite` и т.п.
3. `Navigation.tsx` — красный квадрат-плейсхолдер заменён на графит-логотип через `next/image`. Hover-цвет навигации `red-600` → `tms-orange`. Текст-лейбл «🏁 TMS Telos» сокращён до «Telos» (TMS уже в логотипе).
4. `Breadcrumbs.tsx` — hover-цвет `red-600` → `tms-orange`.
5. Снят глобальный CSS-ban `*, *::before, *::after { transition: none !important; animation: none !important }` из `globals.css`. Это ArtLine-наследие мешало heartbeat-индикатору на live-моторах в EngineCard и любым button-transitions в навигации.

## Что НЕ меняется этим решением

- **Functional severity colors остаются прежними.** `bg-red-500` для violation, `bg-amber-500` для warn, `bg-blue-500` для info — это семантика статуса, не бренд. Менять на оранжевый нельзя: пользователь не должен путать «нарушение» с «бренд-акцент».
- **Тёмная палитра по-прежнему отложена.** Замена основного фона (тёмный automotive-look из принципов ROADMAP) — это отдельный visual overhaul, не часть branding-fix. Текущий decision `2026-05-10-defer-dark-theme-until-branding.md` superceded только в части «брендинг есть — критерий снят», но переход на тёмный фон требует отдельного явного запроса.
- **ArtLine-маркетинг страницы (`/`, `/features`, `DemoCard`) не трогаются.** Они подлежат полной замене в M4 (landing) — точечная замена цветов сейчас была бы временной работой.
- **Дашборды (`/demos/fleet`, `/demos/live-session`, `/demos/anti-cheat-replay`) не используют Navigation.** Они полноэкранные с собственными header-strip. Логотип на них появится отдельным жестом, если станет нужно — пока их header читается как «инструмент инженера», без брендового веса.

## Альтернативы

1. **Перекрасить вообще всё на оранжевый, включая severity.** Отвергнуто: рушит функциональную палитру, делает «critical» неотличимым от «бренд». Operating tool requires semantic colors.
2. **Сразу включить тёмную палитру.** Отвергнуто: полноценный dark mode — разнос на десятки компонентов и charts (Recharts, IMU SVG, GPS-полилиния, `RecorderArea` цвета). Не задача branding-fix, отдельный M4-уровневый scope.
3. **Логотип через `<img>` без `next/image`.** Отвергнуто: `next/image` даёт priority hint и SVG-оптимизацию; стоимость минимальна, выгода реальна.

## Последствия

- ROADMAP «Открытые вопросы → Критичные → Брендинг Telos» закрыт.
- Decision `2026-05-10-defer-dark-theme-until-branding.md` superseded в части критерия «жду брендинг»: брендинг получен. Тёмная палитра остаётся отложенной по новому критерию «отдельный visual overhaul, не приоритет M3».
- При работе над фичами M3+ можно использовать `tms-orange` для брендовых акцентов (выделить активный таб, primary CTA, фирменный chevron в header), но не для severity-индикаторов.
- Если в `presentation.md`-проекте появится bracket-цвет (например для warn/info-вариантов в фирменном стиле) — добавить в `@theme` отдельным кругом, не сейчас.
